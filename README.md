## 🛠 3. 使用方式

請遵循以下步驟來正確載入資料：

1. **準備資料**：
   將下載的 `Sales_csv.csv` 進行解壓縮，並記下該檔案在您電腦中的**完整路徑**（例如：`D:\Downloads\Sales_csv.csv`）。

2. **開啟範本**：
   雙擊開啟 **「我的作品.pbit」**。

3. **連接資料來源**：
   開啟後，Power BI 會自動跳出 **「資料來源顯示視窗」**。
   * 在視窗中的路徑輸入欄位，貼上您剛才準備好的 `Sales_csv.csv` 路徑。
   * 點擊 **「載入」** 或是 **「確認」**。

> [!IMPORTANT]
> **小提醒**：請確保路徑包含副檔名 `.csv`，否則 Power BI 可能會無法正確讀取資料。
